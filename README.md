# RPGMZ-ARTM_EnemyAsActorSpriteMZ
アクターを敵として出現させるMZ専用プラグイン
- **プラグインリスト**
  - ARTM_EnemyAsActorSpriteMZ.js
    - アクターを敵として出現させるMZ専用プラグイン本体
  - ARTM_EnemyAsActorSpriteMZ_DAMZPatch.js
    - 「砂川赳様作 NRP_DynamicMotionMZ」との併用導入パッチ
  - ARTM_EnemyAsActorSpriteMZ_TMZEHBPatch.js
    - 「Ruたん様作 TorigoyaMZ_EnemyHpBar」との併用導入パッチ

- **補足事項**<BR>
  - 本プラグインは敵用SVアクター画像不要の作りとなっておりますが、競合がやや高いためパッチを随時更新中です。
  - 上記競合が気になる方は、敵用SVアクター画像を使用する下記プラグインのご使用がお勧めです。
    - 姫HimeWorks様作 [MV用 [SV]戦闘キャラ(アクター)を敵として使用できるプラグイン](https://himeworks.com/2015/11/sideview-enemy-actors/)。<br>
    - Rexal様作 [敵にアクター用の[SV]戦闘キャラを使用しアニメーション化できるプラグイン](https://forums.rpgmakerweb.com/index.php?threads/animated-enemies.47991/)
