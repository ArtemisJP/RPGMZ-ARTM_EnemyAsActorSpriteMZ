# RPGMZ-ARTM_EnemyAsActorSpriteMZ
アクター画像で敵を出現させるMZ専用プラグイン
- 改修中の内容
  - エネミー設定の特性とアクター設定の特性が両方適用されていた不具合・・修正済
  - アクター画像の敵を倒した際にエネミー仕様の消滅エフェクトが適用されていない・・・対応中
- 検討中の内容
  - アクター画像の敵に対して簡易的なHPゲージを表示する・・・実装するか検討中
